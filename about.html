About This URL Shortener

This URL shortener service was created to provide a simple way to shorten long URLs. The project is built using Node.js and Fastify, and it includes features like:

  • Shortening long URLs
  • Ensuring URLs start with "https://"
  • Preventing the use of blacklisted terms
  • Providing an API for programmatic access

We hope you find this service useful!

API Documentation

Our URL shortener service provides an API to allow programmatic access to the URL shortening functionality. Below are the details:

Shorten URL

Endpoint: POST /api/shorten

Request Body:

{
  "originalUrl": "https://example.com"
}

Response:

If successful:

{
  "shortUrl": "https://dotlink.glitch.me/r/shortenedCode"
}

If the URL contains a blacklisted term:

{
  "error": "URL contains blacklisted term/phrase"
}
